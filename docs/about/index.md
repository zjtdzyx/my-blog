# 关于我

## 小学

我出生于湖北襄阳，来自一个普通的家庭，在高中之前自我意识都是混沌的，感觉就是遵循自己的本能，跟随父母、学校老师的指示按部就班的向前走，没有什么梦想，也不知道自己活着是为了什么。

小学6年于我而言是非常快乐的时光。依稀记得，自己和父母住在一个很大的小区里面，在不上学的时间里，和小区的其他孩子玩就是我最开心的事情。我们在周末的时候，经常会成群结队地在外面玩各种游戏，警察抓小偷、夺宝、十二点、老鼠偷油...... 那个时候最喜欢玩的就是“警察抓小偷”了，因为玩这个游戏的人很多很多，可能有10几号人，而且里面我是最小的，基本上都是跟在那些大孩子的后面。

因为年龄最小，所以发育没有其他人好，跑步能力也没有其他人强，但是我不服输，追不上我就一直追，直到把对方累趴下为止哈哈哈。那段时间非常的无忧无虑，我们甚至发明了一种叫做“西游记”的扑克牌玩法，玩得不亦乐乎，一玩可以玩几个小时，其乐无穷。小学的前几年基本上都是这样度过的，基本上和其他大孩子一起玩各种各样的室外游戏就是我生活的乐趣，他们就是我的全世界。

可是，不知道从什么时候开始，小区里开始有人玩起了游戏，慢慢的，外面的人越来越少，晚上“警察抓小偷”开始聚不齐人，玩“西游记”的人也在慢慢变少，小区没有以前那么热闹了。大家开始从外面疯变成在别人家里疯，我们开始聚集在一个又一个人的家里，看一个人玩电脑。

那个时候玩的比较多的是 CF 和英雄联盟，我们边看一个人玩游戏，边讨论哪个英雄更厉害，剑圣要不要出肉，武器大师配不配超神，哈哈哈。当时很羡慕那些可以在家里玩游戏的人，于是我自己用家里的一台老式电脑，也下了英雄联盟，也下了 CF，但是因为配置太差了，所以进入游戏的时候非常的卡，不过进入游戏后还是可以正常玩的。

那个时期，CF 的挑战模式、英雄联盟的 5v5 成为了我的全世界。当然，我还玩了其他的游戏，和小学班里的同学一起玩了 4399 里面的《造梦西游》、《奥拉星》，有了手机之后，还玩了《球球大作战》《部落冲突》。

影响很深刻的是，那个时候的我还在变声期，玩《球球大作战》的时候，不知道为什么有很多的粉丝，而且打着打着就打上超神了。有一次游戏中，我的粉丝看我玩游戏，说我的声音很好听，是不是长得也很漂亮。我当时一下子就懵了，我一边操作和队友打配合，一边用语音回复那个粉丝：“我是男的！”然后一下子就爆了，我的队友似乎也懵了，粉丝也懵了，都在表示不可思议，哈哈哈。

这就是我的小学——前半阶段，和小区里所有的孩子每天疯来疯去；后半阶段，沉迷游戏无法自拔。

## 初中

上初中之后，我收敛了很多。

影响很深刻的是：由于我小学以为英语只是和品德一样的副课，所以几乎没有学习。刚上初一的时候，连字母有多少个我都不知道。于是初一最开始几次的英语考试都是不及格，倒数第一。

然后我的班主任就叫我家长来学校，我爸说我天天拉一群人在家打游戏，巴拉巴拉的……然后我就一直在挨骂。当时哭得天崩地裂，觉得在其他人面前被说成这个样子非常没有面子。

然后我就下定决心要好好学习！可能是因为我不太服输吧，英语很快就提上来了，在初一上学期的期末考试也考了80来分，感觉跟上大家的节奏了。

学习的事情就告一段落了。

在初一下学期，我无意中看到朋友在看一本书，我很好奇，就问他还有没有，就找他借了一本。仔细一看，叫做《龙族》。那个时候的我还不知道，翻开这本书将意味着什么。那个时候一次好奇的翻阅，让我后来很多年都因为《龙族》的“但为君故”而耿耿于怀。

不知道是我正处于青春年少多愁的时期，还是怎么样，我遇到《龙族》后，一发不可收拾。从早看到晚，那个时候梦里面都是我在卡塞尔学院拳打楚子航，脚踩凯撒。

我最喜欢的情节其实是龙一刚刚开始的时候，龙族世界的开幕。路明非还是一个普通的男孩，暗恋着触不可及的陈雯雯。我也不知道我为什么喜欢这段情节，到现在我都不知道我为什么喜欢，但我就是喜欢。

可能是因为我也是一个衰小孩，我也曾经暗恋过触不可及的女孩吧。

很快，朋友手上的《龙1》和《龙2》就已经被我看完。我自己跑去新华书店买了《龙族3》的上中下——天啊，真是太精彩了！我之前感觉我的世界就是我住的小区，顶多加个学校。但是在读《龙3》的时候，我看到了另一个国家。我沉浸在了日本独特的、和中国不一样的文化之中。我突然意识到：原来世界可以这么大，原来我生活的地方这么小。

在另一个国家文化的冲击下，在《龙族3》的沉浸中，我跟随着路明非的脚步，来到了蛇岐八家，见到了皇——源稚生，遇见了上杉绘梨衣，来到了高天原。有大哥帮我处理其他杂事，我只需要陪伴绘梨衣就好。我带她走遍了我能够想象的所有地方。我慢慢开始怀疑起自己对学姐的情感。我的心底深处有一个声音在不断地回响：**“放弃诺诺吧，她不属于你。她是凯撒的，凯撒喜欢她，她也喜欢凯撒。绘梨衣是你的女孩，她的世界只有你。”**

于是我背叛了组织，做了一个决定：送绘梨衣回家。我站在山上，夕阳在渐渐落下，耳边是海浪冲击礁石的声音。我看着绘梨衣在的火车渐行渐远，心中的女孩也离我而去。我简直进入到了路明非的身体里面，直到绘梨衣走了，我才惊觉——我只是一个读者，一个书外世界的人，他们只是书中的人物，我们不在一个维度。继续看着《龙3》，继续跟随路明非。我为“无天无地之所”——东京塔中猛鬼众的首领王将，还有蛇岐八家的首领橘政宗的对峙而紧张；为源稚生的负刀入场而崇拜；为橘政宗和樱的陨落而难过；为疯狂的风间琉璃而震惊；为恶魔般不死的王将而恐惧。

在一个似乎永无止境的夜晚，我看完了《龙3》，躺在床上，很快就睡了过去。梦中，我仿佛是一个上帝。日本决战在我眼中上演：上杉越那个破坏天地的黑日，源稚生和源稚女的宿命对决，王将就是橘政宗，王将杀掉了绘梨衣，变成了白王，路明非的痛苦，苍穹之上白王与路明非的弑杀，“达摩利斯之剑”的审判。我感觉我不知道度过了多少个世纪，这场战斗在我的梦境中一遍又一遍地重复。当我第二天醒来的时候，我已经察觉不清楚——什么是现实，什么是虚幻了。

时间很快来到了初三。

由于《龙族》的原因，我一下子爱上了小说，整个初二期间都在看小说。《龙族》完本后，我开始看《凡人修仙传》《斗罗大陆》《斗破苍穹》……由于小说的沉迷，让我的成绩退步了不少。在初三第一次大型考试中，我在班里的排名下降了30多名，班上一共50来号人。于是呼，我又被劈头盖脸地骂了一顿，不得不放弃我心爱的小说，开始投身于初三的学习、中考的准备中。

那一段时间，几乎全部的注意力都在学习上。我和小区里的那些孩子渐行渐远。他们都比我大，但因为成绩差，最后大多都去了中专。

我很害怕。我害怕和他们一样上中专。初中的时候，因为我们小区的孩子都在这个学校上初中，他们都比我大，基本上我在学校都有人罩着我，所以我觉得我可厉害了，没有人敢欺负我。可是当我来到初三，他们都去了中专之后，我害怕了。

我开始疯狂地学习。

在我不知道怎么坚持下来的这一年中，我的成绩稳定进入了前列。后来也仅仅因为**0.1分**的差距，取得了襄阳四中在我们学校**最后的一个录取名额**。

我马上就要进入一个高度竞争的百强中学，我不得不告别我的游戏、小说，我和陪我长大的朋友们诀别……

## 高中

### 高一-高二

我记得我进入襄阳四中的那一年，刚好湖北省文理双状元都是襄阳四中的，一时之间，风头无两。可是，骄傲的同时，伴随而来的也是深深的恐惧。

在我进入高中之后，我开始了住宿生活。军训期间，最让我印象深刻的就是：我们在下面军训，对面的高三楼下课打铃时，大家都脚步很快地去上厕所，脚踩在地板上，声音特别大，一整栋楼都是那种“咚咚咚”的声音。我害怕极了，那个场景给幼小的我带来了极大的心理震撼。

很快，军训过去了，开始正常上课。很好，不出所料，我果然跟不上。不仅学号是63（班上总共66人，学号是按入学成绩划分的），而且学习也跟不上。几次数学周考，考完后班主任（数学老师）都会把我拉出去谈话，告诉我说我拉低了班级的平均分……

那段时间，我非常自卑。我的注意力都在学习上，可是无论如何都学不好。月考的班级排名最好的也堪堪在40名左右。那些班级前几、年级前几百的同学，在我眼中，就像高高在上的神仙——可望而不可即。

很快，疫情发生了。高一下一整学期，我们都在家上网课。网课期间，我又忍不住去打游戏，但是一局英雄联盟至少40多分钟，太耗时间了，我只能下定决心彻底戒掉。为了学习，我开始把注意力放在学习上。每天早上自己用电饭煲热包子，然后打开手机进入腾讯会议，开视频，自习……

但让人失望的是，网课期间我们根据选课划分了新的班级，我的学号变成了47。然后网课的每一次大型考试，我的成绩都是垫底，所有科目都堪堪及格，我难过极了。

不过，不知道为什么，返校后的第一次大型考试，我竟然考到了班级前十！明明网课期间都是60多名的我，在那个瞬间简直不敢相信自己的眼睛。我在高兴的同时，也很害怕：是不是大家网课期间没有好好学，所以返校之后成绩还没恢复，等他们一赶上来，我又要垫底？

带着这种恐惧，我发疯似地继续学习。直到又经历一两次月考，我的成绩依旧维持在班级前十名，年级排名在700多左右，我才意识到——原来我的学力真的提升了！

可能是因为上高中后，我一直专注于学习，加上因为自卑经常锻炼身体，当我意识到自己不再是那个糟糕的差学生时，自信一下子充斥了整个身体。直到后来我意识到有同级的女孩频频把眼神停留在我身上，我才发现，原来我已经变成了一个很优秀的人——无论是学习、气质，还是身材、品行。

在我人生最得意的高二阶段，我喜欢上了班里学号第一位的女孩子。她虽然长得没有那么漂亮，但她的学习真的很好，学号第一、成绩第一。在我眼中，她是一个完美的存在，我自动忽略了她其他所有的不足。每次学习遇到困难，我会想：如果是她，她会怎么去思考？

很幸运的是，命运让我们坐在了一起。刚开始我紧张极了，不过后来也就习惯了。离她近了之后，才慢慢去掉了她的“滤镜”，才发现她并不是一个话少、孤傲的女孩。她的话很多，古灵精怪的，而且非常喜欢戏弄我。在我写某门课作业的时候，她看到后也会写，然后写得还比我快，还特地告诉我：“你看，我比你写得快！”小考的时候，我们也在比拼速度，谁都不服谁，结果是我被全方位虐。

那个时间段，我非常抑郁。我感觉我什么都比不上她，我好胜心很强，但就是比不上她。无意识中，我对她的爱慕慢慢减少，友情的含量在不断提高……

我们坐在一起很多个月，从高二上学期末到高二下学期中旬，都是跟她一起。那个时候的几个月，就好像一辈子。

后来，发生了一件让我非常愤怒的事情：我和她几乎绝交。她又一次戏弄了我，但这次我真的无法接受。她组织好了我周围的人，在我的眼镜盒里面放上了一张纸条和一个破破烂烂的金色小鱼钩。纸条上写着：“同学你好！xxx时间可以在xxx等我吗？我有话想跟你说。这个小鱼钩就是我们的信物。”

我当时以为有女孩要跟我表白，我害羞极了，不过也很开心。毕竟没有人不喜欢被人表白。于是我在指定时间整理好仪容仪表，带着那个破鱼钩，在指定地点等着……可是随着时间推移，教学楼里的人渐渐散去，最后除了我，空无一人。我突然意识到：我可能被人耍了。

愤怒涌上心头，我下了教学楼，在操场连续跑了五圈后才回寝室。

第二天，她告诉我是她弄的，而且我周围的所有人都知道这件事。我得知这个消息后，脑子一下子懵了。我回想起自己这些天偷偷摸摸的动作，暴露在他们眼中，是不是就跟一只滑稽的猴子一样呢？

极大的羞耻和愤怒席卷了我，我跟那个女孩绝交了。

接下来发生了一系列我至今都觉得不可思议的事：我跟我最好的朋友倾诉这件事，表示我无法接受继续跟那个女孩坐在一起，想和他换座位，他同意了。结果没过几天，他们两个在一起了。

我知道后，非常愤怒。我不知道我愤怒的真正原因，是因为我觉得最好的朋友背叛了我，还是因为我不甘心？我现在依旧不知道。

我只记得，我把我朋友叫了出来，质问他为什么会这样，为什么我告诉你了她是如何侮辱我，你还会跟她在一起？他告诉我说：“跟我没有关系。”

总之，我跟他们两个绝交了。突然间，我失去了最好的朋友和憧憬的女孩。

印象中，那之后的几个月里，我一闭上眼，脑子里浮现的都是这些事。我不得不强迫自己将注意力放回学习上。匪夷所思的是，我的成绩反而进步了，从年级500多名，进步到400多名。（在襄阳四中，年级一共2000多人，按照历届高考，年级前700是985水平，前1200是211水平。）

就这样，高二结束了。

### 高三

高三又重新划分了一次班级。我选择的是“物政生”组合，这个组合在年级里只有两个班，也就是说，这是一次完全打乱再重组的分班。虽然大部分同学还是之前熟悉的面孔，但感觉一切又都变了。

高三前半段过得很平静。我还是按照之前的节奏学习，假期回家会悠闲地看小说。对了，自从高一网课那段时间把游戏戒掉之后，我高中时期的娱乐方式就彻底变成了**日漫**和**小说**。一旦我对某样东西产生兴趣，我就会特别沉迷进去。

我记得高二那会儿，我几乎把所有比较出名的日漫都看完了，也读了很多让我至今记忆犹新的小说，比如《十宗罪》、《三体》、《诡秘之主》、《万族之劫》……回想起来，那个阶段的我真的非常充实：在学校的时候，专注学习；放假的时候，专注阅读，沉浸在小说和动漫的世界里。那时候我真的以为，未来还很远，还可以永远这样无忧无虑下去。

可是，好景不长。

高三上学期的后半段，我突然发现——**无论我怎么努力，成绩都不再提升了。\**日常学习、作业训练、周考、月考……给我的感觉就是，我好像撞上了一个\**看不见的天花板**。为了突破这个瓶颈，我开始对自己提出更高的要求，强迫自己更刻苦、更拼命。

但我把自己逼得太狠了。

一个小小的偶然事件，导致我接下来的几年都陷入了痛苦的深渊。

那是一个普通的中午，我在寝室午休。可是那天，我**没有睡着**。对我来说，中午的那一觉非常重要，只有午觉睡得好，下午才能保持专注和学习效率。而那天的失眠让我极度焦虑。

第二天中午，我告诉自己：“一定要睡着，一定要睡好。”结果越是这么想，越睡不着。不仅睡不着，而且脑子还开始了胡思乱想。到后来，**我的身体似乎形成了某种“记忆”**——只要一到午休时间，一躺在床上，我的大脑就会自动高速运转，完全停不下来，彻底失去了控制。

这种情况一直持续，不管是在学校还是回家过寒假，午觉的时候总是睡不着，脑子里乱成一团。最开始只是午觉，后来发展成**全天脑子都在不停地运转**，各种杂乱的声音在脑海里回响，极大地消耗了我的精力，让我身心俱疲。

我感觉我的精神状态和身体状况发生了一种很奇怪的变化。但我不知道该怎么让它停下来。恐惧、焦虑像野草一样在我内心蔓延，高三下学期乃至整个高考期间，我始终都处在这种状态下。

好在那个时候，课程已经全部学完。我只能凭借之前积累下来的扎实基础，以及各科训练出的思维模式，**硬撑着维持住成绩**。但过程非常痛苦。

我不知道有没有人能够理解那种感受——**你的大脑不受控制地胡思乱想，焦虑和恐惧从早到晚撕扯你，精神和身体都在被消耗**，你感觉自己已经不是“自己”了，只是在本能地往前撑。

最终，高考的结果也可想而知：一塌糊涂。我没有去查年级排名，但从最后录取的大学来看，估计也就是年级1000~1200名左右。

对了，高三还发生了其他一些事。比如我们家搬了新家，我也看完了《龙族》中的《但为君故》。一边被现实压得喘不过气，一边又被幻想拯救着。

我的高中生活，就这样结束了。

## 大学

高考结束后，我以为自己终于可以好好休息了，精神和身体也该慢慢恢复了。结果完全不是那样。

在高三下学期那种强度的强化下，我的身体和大脑已经形成了一种**病态的“记忆”机制**：即使高考结束，即使没有人再逼我、生活也没有压力，**恐惧与焦虑**依然如影随形，侵蚀着我的精神状态。

我每天都在被消耗，精力不知被谁偷走，大脑像是被施了诅咒。

慢慢地，这种异常开始扩散到了夜晚，我的睡眠也出了问题。我的整个人仿佛进入了一种持续的“失常”状态：神经紧绷、睡眠紊乱、认知下降。最严重的时候，我竟然连手机都不会用了。印象特别深的一天，我拿起手机，看着屏幕上一堆熟悉的 App 名字，明明知道它们是什么，可大脑就是**识别不了、理解不了**。

那一刻，我彻底意识到：我可能得了非常严重的精神问题。

这种状态从高三下持续到了大学前两年，我感受到的不仅是身体和精神的消耗，还有生活方方面面的崩坏——人际关系变糟、不愉快的事情层出不穷，甚至有一度，我觉得我的人生就要在这里终止了。

那个时候，绝望是一种日常。我尝试过向朋友倾诉，但他们似乎并不真正关心；我也向父母求助过，但他们没办法理解“精神问题”这种事。我渐渐明白：**我只能靠自己。**

但或许真的是“天无绝人之路”。

到了大一上学期的寒假，我开始意识到，这可能是一种“精神肌肉记忆”型的问题。它不是一朝一夕形成的，而是在长期焦虑和紧张的刺激下形成的。所以，我试着用“新刺激”去覆盖旧记忆。

我找来了一本小说——《夜的命名术》。假期里，我几乎全天都在专注地读它，把自己沉浸进去。我的想法很简单：我的世界里，只剩这本书。

奇迹发生了。

书看完之后，我感觉自己明显好了很多。从那之后，我不断强化输入，不停给自己灌输新的内容、新的思维、新的情绪，用这种方式去一点点冲淡旧有的焦虑与恐惧。

这个过程整整持续了一年多。到了**大二结束的时候**，我终于可以说：**我基本上恢复了一个“正常人”的状态。**

从高三下学期，到大二结束，我用两年半时间穿越了人生的最低谷。

其中一年多，是我自己摸索出来的“自我治疗”。这一年中发生了太多不愉快的事，说实话，现在我甚至都想不起来了，就像是**大脑为了保护我，自动封存了那段记忆**。也好，就让它沉在心底吧。

到了大三，我终于觉得自己可以重新开始了。结合之前的兴趣和探索，我决定从**前端开发**切入，作为我正式进入技术行业的起点。

从那时起，我开始系统地学习前端，构建自己的技术体系，也尝试参与一些项目，逐渐找到了掌控生活的节奏。

最近，我刚刚从midea离职。这不是一次仓促的决定，而是我**主动为自己争取时间与机会**。接下来一段时间，我会全身心投入到暑期实习的准备中......

说实话，大三之后也发生了很多事，但因为太近了，我暂时不太想说出来。总体上，不是特别顺利，但现在的我，至少拥有了一些**勇气和信心**。

我终于觉得自己可以说出这句话：

> “我想掌握自己的命运，去追逐我的梦想。”
